<h1>Training Deep Neural Networks - Study Notebook</h1>

<p>
This repository contains my study implementation of concepts from 
Chapter 11: <strong>Training Deep Neural Networks</strong> from the book 
<em>Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow</em> by 
Aurélien Géron.
</p>

<p>
The purpose of this project is to reproduce and understand important
techniques used to successfully train deep neural networks using
TensorFlow and Keras.
</p>

<hr>

<h2>Repository Contents</h2>

<p>The main notebook included in this repository:</p>

<pre>
11_training_deep_neural_networks.ipynb
</pre>

<p>
The notebook explores several challenges that occur when training
deep neural networks and presents techniques used to solve them.
</p>

<hr>

<h2>Topics Covered</h2>

<ul>
<li>Vanishing and exploding gradients</li>
<li>Weight initialization strategies</li>
<li>Activation functions (ReLU, SELU)</li>
<li>Batch Normalization</li>
<li>Gradient clipping</li>
<li>Optimization algorithms</li>
<li>Learning rate scheduling</li>
<li>Regularization techniques such as Dropout</li>
</ul>

<p>
These techniques help improve training stability, convergence speed,
and overall neural network performance.
</p>
